📄 파일 도구 최종판

① Markdown 파일 → DOCX로 변환

선택한 MD 파일을 각각 개별 DOCX 파일로 변환합니다.

여기를 클릭하거나 MD 파일을 드래그하세요

② 여러 DOCX 파일 → 하나로 병합

선택한 DOCX 파일들을 순서대로 합쳐 하나의 파일로 만듭니다.

여기를 클릭하거나 DOCX 파일을 드래그하세요

③ 파일명 일괄 변경 (ZIP으로 다운로드)

파일을 선택하고 새 이름 규칙을 정하면, 이름이 변경된 파일들을 ZIP으로 한번에 다운로드합니다.

여기에 모든 종류의 파일을 드래그하세요

변수: {{num}} (숫자), {{title}} (나머지 제목), {{ext}} (확장자)
진행 상태가 여기에 표시됩니다.
